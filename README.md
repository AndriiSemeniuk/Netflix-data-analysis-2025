## 🧐 Key Business Insights
Based on the data visualization, the following trends were identified:

* Massive Scale of Engagement: The analysis covers over 144.8M total views, highlighting the vast global reach of Netflix's top-tier content.

* Content Performance (Movies vs TV): There is a clear distinction in how audiences consume different formats, with specific TV franchises driving more long-term "hours viewed" compared to standalone movies.

* Franchise Dominance: Titles like Squid Game continue to show significant "staying power," maintaining high engagement levels even months after release.

* Period Trends: Interactive analysis of H1 vs H2 periods shows how seasonal releases impact overall platform growth and retention.

## 🛠 Technical Implementation
* Platform: Tableau Desktop & Tableau Public.

* Interactivity: Integrated dynamic period filters (Dropdown) to allow users to toggle between half-year data.
